🎮 Animal Battle Arena

A modern and animated Tic-Tac-Toe style game where animals battle instead of X and O!

🐾 Features:
- Multiple animal matchups (Cat 🐱 vs Dog 🐶, Lion 🦁 vs Deer 🦌, etc.)
- Dynamic theme switching every round
- Player name customization
- Highlighted turns and scoreboard
- Smooth animations and win effects
- Snowfall background for immersive experience

🚀 Built with:
- HTML
- CSS
- JavaScript

Play it online and challenge your friends!
